# TotalNFL-app

Anroid application for my TotalNFL server app, app displays predictions for NFL games.

https://totalnfl-server.herokuapp.com/api/v2/predicted-reg-matches/

Used technologies:
- Kotlin
- Dagger-Hilt
- Retrofit
- RxJava
- ViewModel
- Jetpack Navigation
- Databinding
